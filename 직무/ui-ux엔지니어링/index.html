직무내용

UI/UX엔지니어링은 사용자의 이용 행태와 트렌드 및 기술 환경을 분석하고 새로운 사용자 경험(UX) 모델을 제시하여 이 를 현실화 시킬 수 있는 UI/UX 계획, 실 사용자 리서치, 사용자 인터페이스(UI) 기획, UI 아키텍처 설계, UI 디자인, UI 구현, 프로토타입 검증을 거쳐 가이드를 제작하는 일이다.
