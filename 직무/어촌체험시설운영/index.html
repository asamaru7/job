직무내용

어촌체험시설운영은 어촌의 다양한 자원을 이용하여 체험관광 사업에 필요한 기반시설, 편의시설, 체험시설을 개발하고 운 영ㆍ관리하는 일이다.

능력단위

어촌ㆍ어업의 다원적 기능 파악 수준 : 2

정의
어촌ㆍ어업의 다원적 기능 파악이란 바다생태 환경을 이해하고 어종과 수산물을 보존하기 위하여 친환경 어구ㆍ어법의 활용, 바다환경 오염방지, 그리고 어촌ㆍ어업의 기능을 다각화함으로써 지속가능한 어촌ㆍ어업의 발전을 도모하는 능력이다.

  • 바다 어종 보호하기
  • 친환경 어구ㆍ어법 활용하기
  • 바다환경 오염방지하기
  • 수산자원 보호방법 파악하기
  • 어촌ㆍ어업의 다양한 기능 파악하기
